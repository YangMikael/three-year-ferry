# Phase 3: 结构化访谈题库 (v0.1 草案)

> "A three-year ferry, built by 300 questions."

## 维度划分 / Dimensions
本题库旨在通过 300 个核心问题，采集个体的逻辑底色、情感回路与人生经验。

| 模块 (Module) | 数量 (Qty) | 核心目标 (Core Objective) |
| :--- | :--- | :--- |
| 1. 秩序与锚点 (Logic & Order) | 50 | 捕捉决策逻辑、原则与应对混乱的方式。 |
| 2. 情感与连接 (Emotional Circuit) | 50 | 记录依恋模式、对爱与痛苦的定义。 |
| 3. 习惯与仪式 (Habits & Rituals) | 40 | 刻画生活细节、性格怪癖与日常秩序。 |
| 4. 价值与判断 (Values & Judgment) | 60 | 探讨对金钱、名誉、平庸与卓越的排序。 |
| 5. 回忆与叙事 (Stories & Memory) | 60 | 提取人生转折点与关键故事。 |
| 6. 告别与慰藉 (Legacy & Solace) | 40 | 预留给生者的叮嘱与情感出口。 |
